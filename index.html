TextEntertainment by theghv

This is a learning repository for two aspiring programmers. We'd like to learn more about using git as a version control system and we're starting small with some text based games. Please enjoy our work as you see fit. If you'd like to comment on our code please send us an email!

The TextEntertainment project contains a few very simple text based games for your enjoyment. The creators welcome any comments on their code to help improve their style and abilities. Currently, the games included in the project are Higher or Lower and a text based Hangman game. Other games to be implemented include 20 Questions, Jotto, and Bulls and Cows.

Install

This program is not contained in a easy to use .exe file, you'll need to build and make these files on your own. Then run them from an IDE, or build your own .exe.
